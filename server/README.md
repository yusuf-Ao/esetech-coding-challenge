## Description

NEST.js server code for note taking app. Postgress was used for data storage.

[API Postman Collection](https://documenter.getpostman.com/view/17432527/2s9YypEiHS)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
